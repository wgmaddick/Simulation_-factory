# Simulation_-factory

Streamlit multipage app:

- **Match Report Audit** (`app.py`) — upload rugby telemetry CSV and review player standards.
  Alerts answer when / where / why (optional columns: `observed_at`, `team`, `position`, `venue`).
- **Factory Floor** (`pages/1_Factory_Floor.py`) — live production tick simulation with
  when / where / why alerts (`simulation.py`).

See `AGENTS.md` for run commands. A related Data Center Operations app lives in `datacenter_ops/`.
